# CryoXS-Engine
Multi-Super and Hyper Engine that tries to match a supercomputer's power.
What is a Super and Hyper Engine?
-Super Engines are 100+ engines that focus on raw power and a Hyper Engine is 50+ engines that focus on speed.
There will be three Super Engines, Metanus, Titano, and Cryo, and five hyper engines with no specific names however the group of hyper engines will be called Hypnus.
